# Bias-Aware-News-Aggregator
Bias-aware news aggregator that analyzes articles using NLP and ML to classify content as biased, unbiased, or neutral. Built with Python and Flask, it integrates APIs and sentiment analysis to provide balanced news insights and improve media consumption

## Future Improvements
- Improve model accuracy  
- Add more bias categories  
- Real-time news dashboard  
- User personalization  

## Use Case
Helps users identify biased news and make informed decisions by providing transparent analysis.
